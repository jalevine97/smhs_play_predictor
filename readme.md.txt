# SMHS Play Predictor Prototype

This is a prototype Streamlit app that predicts the play type (RUN or PASS) for SMHS games using historical data.

## Project Structure
- **app.py:** The main Streamlit application.
- **requirements.txt:** Python dependencies.
- **README.md:** This readme file.
- **data/sm_cleaned_with_correct_fields.csv:** Your dataset (make sure to place your CSV file here).
